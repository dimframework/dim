# Security Policy

## Supported Versions

We currently support the following versions of Dim Framework with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Dim Framework, please do **NOT** open a public issue.

Instead, please report it privately via email to:

**Nuradiyana**  
Email: `nooradiana@gmail.com`

### What to Include

Please include as much information as possible to help us reproduce the issue:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Any proof-of-concept code (if available).

### Our Process

We will review the report and work towards a resolution. We appreciate your cooperation in not disclosing the vulnerability publicly until it has been addressed.

Thank you for helping keep Dim Framework secure!
